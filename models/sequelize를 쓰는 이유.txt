SQL문 대신
외부 패키지를 통해 JavaScript 객체를 다루면
데이터베이스에 간편하게 새 요소를 생성하거나 편집하고
삭제하거나 찾을뿐 아니라 연결하는데 편리하다.

데이터베이스는 관계까지 처리하는데 지금은 관계를 처리할게 없음.

Sequelize는 제 3자 패키지(써드패키지)중 하나로
Object-Relational Mapping Library
ORM이다.

백그라운드에서 실제로 SQL코드를 처리하며, JavaScript 객체로 맵핑해
SQL 코드를 실행하는 편리한 메서드를 제공함


Sequelize는
데이터베이스를 다루는 모델을 제공하며, 정의할수 있다. User, Product
모델을 구성하는 데이터, 데이터베이스에 저장될 데이터를 정의하는것
다음은 모델, 즉 사용자, 제품 같은 클래스를 실체화 할
생성자함수, utility 메서드를 통해  모델에 기반한 새로운 사용자 객체를 생성하여 연결된다. 예 ) const user = User.build()

쿼리도 실행할수 있고, 항상 Sequelize로 정의한 모델과 관련이 있음. 예 ) User.findAll()

또 모델을 연관지을수 있는데, User 모델을 Product와 연관시키는것. 예 ) User.hasMany(Product)
